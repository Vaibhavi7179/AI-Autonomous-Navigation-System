# 🚗 AI-Based Autonomous Navigation System

## 📌 Overview

This project simulates an AI-based navigation system that finds the shortest path between two points using the A* algorithm.

## 🎯 Objective

To demonstrate how autonomous systems navigate efficiently while avoiding obstacles.

## 🧠 Concepts Used

* Path Planning (A*)
* Grid-based Navigation
* Obstacle Avoidance
* Visualization using Matplotlib

## 🛠 Tech Stack

* Python
* NumPy
* Matplotlib

## 📂 Project Structure

```
main.py → runs the project  
grid.py → creates environment  
pathfinding.py → A* algorithm  
visualization.py → displays result  
```

## ▶️ How to Run

```
pip install -r requirements.txt
python main.py
```

## 📸 Output

* Start point (Green)
* Goal point (Red)
* Path (Line)
* Obstacles (Black blocks)
